# 📁 市场日记 — 周度回顾系统

## 目录结构

```
~/.hermes/reviews/
├── README.md              ← 本文件
├── 2026-07-03.md          ← 每日复盘
├── 2026-07-06.md
├── 2026-07-07.md
├── 2026-07-08.md
├── 2026-07-09.md
└── weekly/
    ├── 2026-W27.md        ← 第27周周报
    └── 2026-W28.md
```

## 使用方法

### 每日复盘
对我说「复盘」，自动生成当日报告并存入该目录。

### 周度回顾
每周末对我说「生成周报」或「weekly review」，我会：
1. 读取本周所有日度复盘
2. 汇总关键趋势（指数变化轨迹、板块轮动、资金流方向）
3. 输出周报并存档到 `weekly/` 目录

### 手动查阅
- 「看看这周的复盘」 → 列出本周所有日度复盘文件
- 「对比 7/6 和 7/9 的行情」 → 跨日对比

## 复盘覆盖
- 🇨🇳 A股：五大指数、涨跌家数、成交额、板块轮动、资金流向
- 🇭🇰 港股：恒生/恒生科技/国企指数、领涨领跌
- 🥇 现货黄金：XAUUSD 价格、日内走势、驱动因素、关键位
- 🇺🇸 美股（可选）
